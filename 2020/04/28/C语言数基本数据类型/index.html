数据类型

数据类型，打个比方的话，就类似于部队中，前面我们己经给部队中人的起了名字，现在赋予这个权力和管辖范围，那就是军衔。或是通俗的讲，类型相当于模子，变量的定义，就相当于在内存中用模子框出的一块空间。

数据类型说明

我们先给出C语言中数据类型的框架，这篇博客主要来说明C语言基本类型。

我们先来回顾一下变量：

变量的定义格式是：

类型 变量名；

类型决定了变量所占字节的大小

变量名是一段内存取别名

类型+变量名——->生成变量

类型决定变量的大小

数值类型

整型

内存编址的最小单元是字节，不同的数据类型决定变量占据不同大小的字节。

我们在这里给出整型的表示范围:

不同整型的大小

#include <stdio.h>
int main()

printf("sizeof(float) = %d\n",sizeof(float));

printf("sizeof(long double) = %d\n",sizeof(long double));
return 0
}

一个浮点型由三部分组成：符号，阶码，尾数。接下来我们进行解释和表示

其中 S 是符号位，P 是阶码，M 是尾数。

字符类型

基本字符类型

在上面的保险单中存储在计算机内存中，年龄是整型数，保险金额是浮点数。

但是计算机内存中不能存储字符M。

我们通过计算机显示能看到M，但是实际上在内存中存放的不是M，存放的是一个数值。

我们先给出字符M对应的数值是char类型的77，当变量是char类型并且77的时候如果要显示，那么就显示为M。

计算机内存中不能存储字符，存放的是字符对应的数值，只能在显示的时候进行转化，遵循ASCII码转化。这种转化需要一张表，表里记载了转换规则，这张表，就是 ASCII 编码表。该表的本质就是一种字符类型和char类型建立起来的一一对应关系。

字符类型实际是一个char类型，所能表示的无符号是0 ~ 255，ASCII只用到0~127的范围。

基本的 ASCII 字符集共有 128 个字符，其中有 96 个可打印字符，包括常用的字母、数字、标点符号等，另外还有 32 个控制字符。

我们下面说明ASCII表：

我们来通过ASCII表的转化关系打印出来所有的char类型对应的字符类型：

#include <stdio.h>
int main()
{
    for (unsigned char ch = 1; ch <= 128; ch++)
    {
        printf("%3d -->%c\t", ch, ch);
        if (ch % 8 == 0)
            printf("\n");
    }
}

运行结果为：

当把char类型和ASCII码建立一一对应关系之后，我们把char类型也叫字符类型。

内存本质存储的是整型，只不过我们需要转化的时候通过%c向屏幕上打印字符。

计算机在内存中只能存储数值，要么是整数，要么是小数。但是如果要在屏幕上显示，我们知道屏幕最终来自于显卡的渲染，字符类型存储的时候本质上存储的是char类型数值，那么当我们要显示字符的时候，把存储的char类型数值交给printf函数，printf函数通过参数%c把内部的char类型数值对应ASCII字符显示出来。

所有图片，音频，视频等多媒体的类型都是通过显卡预定的规则而转化渲染而来的。

所以本质上所有的信息都是以二进制存储在计算机内部。

我们用UE打开图片文件：

上面存储的十六进制本质上就是存储二进制。

在计算机中，所有数据的存储形式都是以01的二进制形式存储的，我们能通过显示器看到图片是因为把数据发送到显卡，显卡通过预先预定的规则，把二进制数据渲染成图片，视频等其他形式进行显示。

我们在这里强调 字符0和数字0是不一样的：

printf("%d\n",0);  //数字0
printf("%d\n",'0');  //字符0

转义字符

为什么需要转义字符？

#include <stdio.h>
int main()
{
        printf("I like "football"");
        return 0;
}

上面代码会出现编译问题，不能直接在字符串里面打印” “

C语言中” “ 表示字符串和打印的字符串发生冲突，但是我们又想要使用，那么就把原先C语言的意义脱去，及就是使用转义。

转义的前提就是说在这地方已经被用作其他用途，现在上来要使用不同于之前的用途就需要转义。

还有一种例如：

n本身就是字符，但是C语言想实现一个换行的功能就把n转义为\n。

转义字符及其意义

我们来介绍转义字符表示方法和功能：

转义字符就是本身己有一个字符的意义，我们给它另外一个别的意义。

转意字符常起到控制的作用。

字符语义被征用，有新的需求，通过转义实现新的功能。
